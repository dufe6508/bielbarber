# Imagens dos produtos

Coloque aqui os arquivos referenciados no seed (`prisma/seed.ts`):

- `gel-boy.jpg` — Gel Boy (ForceMen, embalagem vermelha)
- `prime-fix.jpg` — Prime Fix (cera modeladora, embalagem preta)

Use imagens quadradas (ex.: 800×800). Os cards normalizam via `object-cover`,
então proporções diferentes funcionam, mas o quadrado fica mais consistente.
